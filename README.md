# Infinity Restaurant Website Clone

A frontend clone project inspired by the Infinity Restaurant website. This project was developed for learning and practice purposes to improve frontend development skills, responsive web design, and restaurant website implementation.

## 🚀 Project Overview

The Infinity Restaurant Website Clone is a responsive frontend project that recreates the design and user interface of a modern restaurant website. The project focuses on building an attractive dining experience through clean layouts, engaging visuals, and responsive design while practicing real-world frontend development techniques.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)

## ✨ Features

- Fully Responsive Design
- Modern Restaurant Website Layout
- Hero Banner Section
- About Restaurant Section
- Food Menu Section
- Featured Dishes Section
- Gallery Section
- Chef Team Section
- Reservation Section
- Contact Section
- Smooth Navigation
- Mobile Friendly Interface
- Cross-Browser Compatibility
- Clean and Organized Code Structure
